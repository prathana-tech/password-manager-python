# 🔐 Password Manager (Python)

A secure and user-friendly password manager built using Python.  
This project demonstrates authentication, file handling, and real-world problem-solving.

---

## 🚀 Features

- 🔐 PIN-based login system (3 attempts)
- 🔄 Security question recovery system
- 🆕 First-time user setup
- 🔑 Strong password generator
- 💾 Save passwords securely
- 📂 View saved passwords
- 🔍 Search passwords by website

---

## 🧠 Technologies Used

- Python
- File Handling
- Random & String modules

---

## 📁 How It Works

1. First-time user sets PIN and security questions
2. Data is stored in a file
3. On next run:
   - User logs in using PIN
   - If PIN is forgotten → recovery option available
4. User can generate, save, view, and search passwords
